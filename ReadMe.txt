h(Art)Beat
----------------------------------------
- Audio Spectrum
- Created through Unity3D and C#
- Uses soundwave algorithms to split sound into an array of float
describing the loudness of each frequency
-----------------------------------------
Playing Songs:
- Comes preloaded with Flume's remix of Lay Me Down
- *Note* Song can only be changed within Unity3D
- Drop songs into Resources/Music
- Go to Audio Source object and set songName to the name of song
  (w/o file extension)
------------------------------------------
Controls:
- Space to pause/unpause
- "M" to show/ hide text
